# DMS Proto — Project Notes

## Active Application: Unified DMS_RB

The application is consolidated into **DMS_RB**.

- **Frontend (active):** `DMS_RB/client/` — React Bootstrap frontend
- **Backend (active):** `DMS_RB/server/` — Node.js / Express / SQLite, runs on port 3000
- **Runtime data:** `DMS_RB/data/` and `DMS_RB/storage/`

All future application development targets `DMS_RB` only.
