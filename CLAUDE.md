# DMS Proto — Project Notes

## Active Application: Bootstrap Version (DMS_RB)

The user only wants to work on the **Bootstrap version** of the application.

- **Frontend (active):** `DMS_RB/` — React Bootstrap frontend
- **Backend (shared):** `DMS_V2/server/` — Node.js / Express / SQLite, runs on port 3000
- **Frontend (inactive):** `DMS_V2/client/` — do NOT modify unless explicitly asked

All future UI/frontend development targets `DMS_RB` only.
`DMS_V2/server/` is shared infrastructure used by both frontends.
